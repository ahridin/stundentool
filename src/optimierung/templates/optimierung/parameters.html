Einstellung der Optimierungsparameter





Hier können sie alle Parameter für die Optimierung einstellen: Die Parameter entscheiden darüber, welche Lösung zur optimalen wird, jenachdem wie stark bestimmte Teile der Zielfunktion gewichtet werden. Außerdem können sie hier einstellen, welchem Solver die Daten übertragen werden sollen.

Bitte geben sie unter dem entsprechenden Punkt für die Paramerter einen Wert ein, der symbolisieren soll, wie wichtig ihnen die Erfüllung dieser Soft Constraint ist.

Unter "Solver" geben sie ein, welchen Solver sie für die Optimierung verwenden wollen.
